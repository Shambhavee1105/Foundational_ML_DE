# Fundamentals of Machine Learning & Data Engineering

This repository contains a collection of essential machine learning, data preprocessing, visualization, and statistical analysis codes.
It is designed for beginners and students who want hands-on practice with core ML algorithms and data engineering workflows.

---

##  Repository Contents

### Jupyter Notebooks

These notebooks include step-by-step implementations, visualizations, and explanations.

* `Clustering_KMeans_Spectral_DBSCAN.ipynb` – Implementation of K-Means, Spectral Clustering, and DBSCAN
* `Data_Visualizations.ipynb` – Exploratory graphs using Matplotlib & Seaborn
* `DecisionTree.ipynb` – Basic Decision Tree model
* `DecisionTree_SVM.ipynb` – Comparison between Decision Tree and SVM
* `Ensemble_Learning.ipynb` – Bagging, Boosting, Random Forest examples
* `KNN_NaiveBayes.ipynb` – Implementation of KNN and Naive Bayes
* `Regression_Linear_Multiple.ipynb` – Linear & Multiple Linear Regression

---

### Python Scripts

Reusable Python modules for fundamental ML and data engineering operations.

* `basic_data_preprocessing.py` – Loading datasets, handling missing values, encoding, normalization
* `basic_statistical_ops.py` – Mean, median, mode, variance, and standard deviation
* `basic_data_visualizations.py` – Histograms, scatter plots, bar and box plots
* `basic_clustering.py` – Basic clustering algorithms with examples
* `basic_ml_algos.py` – Simple implementations of common ML models

---

## Skills Covered

* Data preprocessing & cleaning
* Exploratory Data Analysis (EDA)
* Feature engineering basics
* Supervised ML algorithms (Regression, Classification)
* Unsupervised ML algorithms (Clustering)
* Ensemble learning fundamentals
* Model comparison & visualization
